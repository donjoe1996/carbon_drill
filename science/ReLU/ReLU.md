My understanding on ReLU:  
- In deep learning, one of the core technical idea is RELU, which in plain english actually data transformer. Linearly separable n-dimensional data, can not be perfectly divided because they are linear, there will be no single line that could do this.
- hence, we need to transform the data using data transformer e.g RELU. What RELU does is actually transform linear data to non-linear within range values [0 to positive]. Having these values range helps to classify the data (think about it).
- however, it's worth to mentioned that RELU is not a classifier, it's just a data transformer which prepare the data. The output of RELU transformation will be used by other classifier to make classification.
