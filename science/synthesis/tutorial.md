How to annote pdf
1. Install annotator, go to left pannel -> gear box
2. Find communinty plugins and install
3. Create a new file cmd + N
4. Create an annotate note e.g using markdown
5. --- annotation-target: name of file (in double bracket) ---
	 **Why this design?**
	The Annotator plugin keeps your **annotations separate** from the PDF file itself:
	- **PDF file** = the original document (stays clean, unmodified)
	- **Annotation note** = stores all your highlights, comments, and notes in markdown format
	- This way your annotations are searchable, linkable, and part of your Obsidian knowledge base
	
	**The problem:** This workflow is complicated!
